Universal rules - for all torrents


Images

IMGUR DOES NOT WORK ON THE 1337X TORRENT INDEX PLEASE DO NOT USE IT

In some sections the requirement of a screenshot is mandatory in others it is appreciated but not required. You will find out what sections require them towards the end of this guide. We have a white list of prefered image hosts. You do not have to use this but if you are planning on using one not on the list, it must comply with the following conditions.

The image hosting site must not be on the ban list. The ban list can be found under the white list. No money making image hosts allowed. Host ads should be no more than 5 but this will not be strictly enforced so long as the ads are responsible. No ads serving malware. Any hosts with malware ads will be added to the ban list straight away.


Advertising & File names


1337x is a file-sharing site we do not support any schemes for uploaders to try and make money.

Banned

  • Video watermarking - Ads or site urls added to video print by encoders & uploaders.
  • Audio watermarking - Ads or site urls added to audio tracks by encoders & uploaders.
  • Urls in folder & file names. (Trial uploader class)
  • Urls in torrent descriptions. (Trial uploader class)
  • Urls in torrent titles. (All classes)
  • Banner ads in torrent descriptions.
  • ANY links to 3rd party sites apart from a text file included in the torrent folder. Rules about this bellow.
  • Links to gambling sites, direct download only sites or anything else unrelated to p2p.
  • Link Shortners/Spam/Surveys.
  • 18+ & First On Net tags.
  • Emojis in torrent titles unless you are VIP rank.

3rd party site links

Trial uploaders are no longer allowed to link to any 3rd party site or offsite places in folder names or torrent descriptions. The exception to this is sites like IMDB, TVDB, goodreads and similar sites used for file information.

You are allowed to include a text file inside your torrent with a link to your site.
If you are including a text file to a 3rd party site then the siteMUST comply with the following,

No excessive ads
Strictly no ads serving malware (You get one chance and one chance only)
The site MUST offer .torrent files and not just direct download links and/or file-locker links. (Exceptions - Server / seedbox / vpn providers, subtitle sites, Imdb/Tvbd or anything similar)
The site MUST be related to file-sharing or file information.
You MUST tell staff if you plan to include a text file link to your site.

ANY TRIAL UPLOADER THAT BREAKS THESE CONDITION'S LOSES ALL LINKING PRIVILEGES - All torrents will be deleted and you may or may not be allowed to continue to upload with no links.
There is no second chances. It is up to you to make sure the site you link to complies with this. If you cannot then do not link to it.

When and if a Trial Uploader is promoted to Trusted Uploader they will be allowed to include links in their descriptions. The linked site still has to comply with the rules above.

Section specific rules


Movie uploads

We require a minimum of one visible screenshot in your description for every movie upload. These screenshots are to be from the video you are uploading. Using the same screenshot for 1080p/720p or any different resolution or codec is not appropriate. Cover/Title posters are appreciated but do not count as a screenshot. Movie uploads must not contain any zip, rar or any other file archive formats. Movie torrents are also subject to the universal rules above.

Tv, Anime & Documentary uploads

These sections do not require screenshots but screenshots are appreciated. When including screenshots they must comply with our universal rules above. Tv & Anime uploads must not contain any zip, rar or any other file archive formats. Tv & Anime torrents are also subject to the universal rules above. If for some unknown special reason you need to use rar/zip's you must get staff permission.

Other Section

Uploads in this section must have a description and not just offsite links. Rar/zips are sometimes needed for certain things like plugins etc, Make sure to get staff permission first. Screenshots are welcome but not mandatory. Historically this section has been a bit spammy - Please don't make it worse.

Applications & Game uploads

Anyone wishing to upload software or games will have to prove they are trusted elsewhere. This will ideally be another torrent site but release blogs and other forms of file sharing services might be accepted. We refuse 95% of software uploader applications so be prepared for a refusal. You will also have to demonstrate you have enough knowledge to check software for malware.

It is the responsibility of the uploader to check software for viruses prior to uploading. If you do not know how to check if a file is clean and what false positives are then either learn first or pick another category to upload in. Sites like Virus Total and Hydrid Analysis are excellent resources. Tools like Virtualbox and VMware are awesome for checking bigger files. You will be expected to demonstrate Staff are not here to check files for you. Staff will however randomly check files of uploaders at all levels. If any are found to have infected files you will be demoted and/or banned.

XXX uploads

We require a minimum of one visible screenshot in your description for every XXX video upload. These screenshots are to be from the video you are uploading. Using the same screenshot for 1080p/720p or any different resolution or codec is not appropriate. Cover/Title posters are appreciated but do not count as a screenshot. XXX uploads must not contain any zip, rar or any other file archive formats.

It is hard to find image hosts that allow porn. Therefore we are a little more forgiving when it comes to the XXX section. Please do not use an image host under the banned list here. We have a white list of preferred XXX image hosts. You are free to use your own host but it must comply with the universal rules above.
